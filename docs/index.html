Mirscope
全身镜

Full Mirror, Full Insight

本地优先的 AI Prompt 全景分析平台。自动采集多 IDE 对话数据， 在本地完成统计、可视化与智能分析，完整映照你的 AI 使用习惯。

下载 Release 查看源码

核心功能

Dashboard 全景看板

使用时段热力图、平台分布、活跃项目与高频 Prompt 短语，一眼看清使用全貌。

Timeline 时间线

按平台与项目浏览历史 Prompt，支持项目画像与 AI 驱动的习惯分析。

Explorer 资源浏览

FTS5 全文检索、多维筛选、收藏与详情，快速定位任意一条 Prompt。

增量同步引擎

启动自动同步 + 文件监听增量更新，内容级去重，数据不膨胀。

AI 智能分析

Prompt 评分、优化建议与项目 Playbook，调用用户自有模型 API。

隐私优先

数据默认仅存本地，API Key 加密存储，无强制云端上传。

系统架构

┌──────────────────────────────────────────────┐
│              React UI (Renderer)              │
│  Dashboard · Timeline · Explorer · Settings  │
└──────────────────────┬───────────────────────┘
                       │ IPC
┌──────────────────────▼───────────────────────┐
│           Electron Main Process               │
│  ConnectorManager · SyncEngine · IPC Handlers │
└──────────────────────┬───────────────────────┘
                       │
     ┌─────────────────┼─────────────────┐
     ▼                 ▼                 ▼
 Connectors      Normalization       Database
 (插件采集)        (标准化/去重)      (SQLite + FTS5)
     │                 │                 │
     └─────────────────┼─────────────────┘
                       ▼
              Analytics · AI Provider

五层解耦：UI → Connector → Normalization → Analytics → AI。新增平台只需开发 Connector 插件。
